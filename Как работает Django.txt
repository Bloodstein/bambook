ПОСЛЕДОВАТЕЛЬНОСТЬ УСТАНОВКИ ПРОЕКТА:
1. python -m venv myvenv
2. myvenv\Scripts\activate
3. django-admin startproject mysite . (обязательна точка)
4. Настройка settigns.py

КАК РАБОТАЕТ ДЖАНГО:
1. Пользователь вводит адрес в браузере - происходит запрос
2. Сервер принимает запрос
3. Сервер отправляет запрос в джанго
4. Джанго получает запрос и считывает из него адрес страницы
5. Джанго проверяет адрес страницы по файлу urls
6. Джанго смотрит название вьюхи и переходит на него, передавая ответ на запрос от браузера
7. Отрабатывается весь код во вьюхе и отдается темплейт (html) и в него переменные
8. Браузер отрисовывает html-файл пользователю.

Запрос поступает в url в папке сайта
Проводит проверку по подходящему url и переходит по его include

url(r'^',include('landing.urls'))

В папке landing переходит в url и проводит проверку по нужному url
По нужному url находит нужную вьюху и переключается на неё

url(r'^landing/', views.landing, name='landing')

В views джанго находит функцию landing и возвращает её по запросу пользователю в браузер
